/*  rogerSmartSql
	SQL result:
	row1 : china, 86, ....
	row2 : Ameriaca, 01, ....
	row3 : England, 02, ....
	auto convert to json:
	[{Country:'china',Code:86, ...},{ Country:'Ameriaca',Code:01, ...},{Country:'England',Code:02, ...}]
	instead need generate object:
	{
		China: {Code:86, ...},
		Ameriaca: {Code:01, ...},
		England: {Code:02, ...}
	}
	so, add keyword 'key' to indicate which sql result colum name is key.
*/
